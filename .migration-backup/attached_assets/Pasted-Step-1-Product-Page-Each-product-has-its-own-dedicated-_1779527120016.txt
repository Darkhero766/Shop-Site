Step 1 — Product Page
Each product has its own dedicated buying page at /s/[shopname]/product/[productid]. When a buyer clicks "Buy Now" on any product card, they go to this page. Show the product images (swipeable), name, price, description, sizes as selectable pill buttons (required if sizes exist), and a prominent "Buy Now" button at the bottom.
Step 2 — Order Details Form
Clicking "Buy Now" opens a full page (not a popup) with a clean form collecting:

Full name (required)
WhatsApp number (required, 10 digits, Indian format)
Full delivery address (required, textarea)
City (required)
Pincode (required, 6 digits)
State (required)
Selected size (pre-filled if chosen on product page)
Special instructions (optional)
Order summary shown on the right side (or top on mobile): product image, name, size, price

Show a "Proceed to Payment" button at the bottom. Validate all required fields before proceeding.
Step 3 — Payment Page
After form submission, show a dedicated payment page with:

Order summary at top (product, size, amount in ₹)
A unique Order ID generated by the system (e.g. ORD-2024-XXXXX) shown prominently
The shop's UPI QR code large and centered and scannable
UPI ID text below the QR with a copy button
Step by step instructions: "1. Scan QR below  2. Pay exact amount ₹[price]  3. Upload screenshot or enter transaction ID"
Two options for payment proof side by side:

Option A — Upload payment screenshot (image upload, accepts jpg/png)
Option B — Enter UTR / Transaction ID manually (12 digit numeric input)


A "Confirm Order" button that only activates after either screenshot is uploaded OR UTR is entered
Save the order to the orders table in Supabase with all buyer details, selected size, address, order ID, UTR or screenshot URL, status = pending

Step 4 — Confirmation Page
After submitting, show a clean success page with:

Green checkmark icon
"Order Placed Successfully!"
Their Order ID (large, copyable)
Summary: product name, amount paid, delivery address
"The seller will confirm your order on WhatsApp within a few hours"
A WhatsApp button that opens wa.me/91[seller_number] with pre-filled message: "Hi, I just placed order [OrderID] for [product name]. Please confirm."
"Continue Shopping" button that goes back to the shop homepage

Design requirements:

Mobile first, single column on phone
Progress bar at top showing: Product → Details → Payment → Done
Each step is a separate full page, not tabs or popups
Smooth fade transition between steps
All buttons rounded-full, purple primary color
Show a loading spinner on "Confirm Order" while saving to Supabase
If UTR already exists in database, show error: "This transaction ID was already used. Please check and re-enter."
Screenshot upload goes to Supabase storage bucket named payment-screenshots (create if not exists)

Also update the seller dashboard Orders tab to show: buyer name, phone, full address, pincode, city, product, size, amount, Order ID, UTR or screenshot link, and status buttons (Confirm / Decline).
